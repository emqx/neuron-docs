# 使用 .tar.gz 包安装

## 下载

根据不同版本及架构下载安装包，例如：

```bash
$ wget https://www.emqx.com/en/downloads/neuron/2.4.0/neuronex-2.4.0-linux-amd64.tar.gz
```

## 安装

```bash
$ tar -zxvf neuronex-2.4.0-linux-amd64.tar.gz
$ cd neuronex-2.4.0-linux-amd64
```

## 启动

执行如下指令启动 NeuronEX：

```bash
$ ./neuron-helper.sh start
```
